Descripcion del Proyecto

Este proyecto implementa la lógica base y algoritmia para una aplicación web
que gestiona usuarios y servicios ofrecidos por estos. Se desarrolla utilizando el patrón de arquitectura MVC para organizar el código de manera clara y escalable.

Tecnologias Utilizadas
Las tecnologias utilizadas en este proyecto son
JavaScript
HTML
Git
